# skew_distribution_python_tests
La repository riporta i codici utilizzati per la redazione dell'elaborato finale del corso di statistica per le tecnologie e le scienze dell'Università degli Studi di Padova. L'elaborato ha titolo "Pacchetti Python per la distribuzione Normale Asimmetrica: analisi comparativa con R e proposta di un approccio integrato".
I codici presenti contengono anche delle parti non utilizzate per la stesura dell'elaborato.
La versione con la spiegazione in inglese sarà disponibile a breve.

The repository contains the codes used to write the final paper for the statistics course for technology and science at the University of Padua. The paper is entitled “Python packages for asymmetric normal distribution: comparative analysis with R and proposal for an integrated approach".
The codes also contain parts that were not used in the preparation of the paper.
The version with explanations in English will be available shortly.
